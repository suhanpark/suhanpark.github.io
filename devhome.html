Lottery Number Picker

Machine learning based algorithm to prove lottery is entirely randomized.
• A Python based software that predicts the future lottery number.
• Version 1 gathered past winning numbers of lottery by web scraping and processed the data in lxml form using Python. The numbers are ranked by how frequently showed up on winning numbers, and the future number combination(s) is predicted using probability statistics.
• Version 2 gathered the data with the same method as version 1, web scraping, then processed the data in arrays using Numpy for efficiency. Then, using LSTM (long short-term memory) of RNN architecture (recurrent neural network) to train models and predict the future numbers using TensorFlow framework.
